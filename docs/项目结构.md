# 项目结构说明

## 根目录

| 路径       | 说明           |
|------------|----------------|
| `backend/` | Python 后端项目目录 |
| `frontend/`| Vue 3 前端项目目录  |
| `docs/`    | 项目文档         |
| `README.md`| 项目说明         |

## backend/

| 路径 | 说明 |
|------|------|
| `app/main.py` | FastAPI 应用入口、CORS、路由挂载 |
| `app/config.py` | 配置（数据库 URL、JWT、统一密码等） |
| `app/database.py` | SQLAlchemy 引擎、Session、get_db |
| `app/models/` | 模型（user 等） |
| `app/schemas/` | Pydantic 模型（登录、Token、UserOut） |
| `app/core/security.py` | 密码哈希、JWT 签发与解析 |
| `app/api/auth.py` | 认证路由：POST /api/auth/login、GET /api/auth/me |
| `app/api/deps.py` | 依赖：get_db、get_current_user |
| `requirements.txt` | Python 依赖 |
| `.env.example` | 环境变量示例 |

后续将增加：综合选股、持仓、数据源适配、定时任务等模块。

## frontend/

| 路径 | 说明 |
|------|------|
| `src/main.ts` | 入口，挂载 Vue、Pinia、Router、Element Plus |
| `src/App.vue` | 根组件，router-view |
| `src/router/index.ts` | 路由（登录、首页）、鉴权守卫 |
| `src/stores/user.ts` | 用户状态、token、登录/登出 |
| `src/api/http.ts` | axios 实例、请求头带 token、401 跳转登录 |
| `src/api/auth.ts` | 登录、/auth/me 接口 |
| `src/views/LoginView.vue` | 登录页 |
| `src/views/HomeView.vue` | 首页 Demo |
| `vite.config.ts` | Vite 配置、代理 /api → 后端、@ 别名 |
| `package.json` | 依赖（Vue 3、Vue Router、Pinia、Element Plus、axios） |

后续将增加：综合选股页、持仓助手页、图表与表格组件等。
