# 调研结论：技术指标扩展

**日期**: 2026-03-22 | **规格**: [spec.md](./spec.md)

## 1. 「均线」口径

- 国内市场常用 **MA5/10/20/60** 指 **简单移动平均（SMA）** 对收盘价。
- **本期采用 SMA**，与多数行情软件「均线」默认一致；若未来支持用户切换 EMA，另开规格。

## 2. MACD 口径

- 采用 **DIF = EMA12 − EMA26**，**DEA = EMA9(DIF)**，**柱 = 2×(DIF−DEA)**。
- **EMA 初值**：建议在实现中固定一种（例如首根 `close` 作为 EMA 种子，或前 N 根 SMA 预热），并以**单元测试黄金样本**锁定，避免与 pandas `ewm` 默认 `adjust` 参数不一致。

## 3. 与复权

- 当前 bar 表 `close` 与 Tushare 拉数口径一致（多为**前复权**或未复权，以项目现有同步为准）。
- **指标是对表内 `close` 序列计算**，不在指标层二次复权；若日后切换复权策略，需**全量重算**指标。

## 4. 依赖库

- 使用 **pandas** 做全序列向量化便于与手工表格抽查；生产路径可对单标的 `Series` 计算，避免巨宽表。

## 5. 性能粗估（非承诺）

- 单标的日线 5～10 年约 1200 行，向量化毫秒级；全市场 5000 标的顺序执行约在数十分钟量级（视机器与 IO），需**实测**后写入运维预期；计划中的分批与日志用于控制风险。
