```bash
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 前端启动

```bash
cd frontend
npm run dev
```

## 验证步骤（实现完成后）

1. 浏览器登录，侧栏进入 **「个人服务」→「个人持仓」**。
2. **建仓**：选择 `600000.SH`（或库中存在的代码），录入数量与价格，保存后「当前持仓」仅一行。
3. **加仓**：同一股票再次加仓，列表仍一行，**详情**中可见两条操作记录。
4. **清仓**：卖出全部数量，持仓从列表消失，**已完结**中出现一条，`realized_pnl` 与手工计算一致。
5. **复盘**：在已完结详情中填写文字、上传一张 PNG 图片，刷新后仍可见；图片请求需带 Token。
6. **胜率**：对若干操作标记「好/坏」，打开汇总区，股票胜率与操作胜率与手工统计一致。
7. **隔离**：换另一用户登录，不应看到上一用户的持仓与记录。
